Ejercicio 22 - Investigación de Propiedades CSS

border-top:

Definición: Establece el borde superior de un elemento. Es una propiedad abreviada que combina border-top-width, border-top-style y border-top-color.

Sintaxis: border-top: [ancho] [estilo] [color];

Ejemplo:

Este div tiene un borde superior de 3px sólido rojo.
border-top: 3px solid #ff6b6b;

border-bottom:

Definición: Establece el borde inferior de un elemento. Es una propiedad abreviada que combina border-bottom-width, border-bottom-style y border-bottom-color.

Sintaxis: border-bottom: [ancho] [estilo] [color];

Ejemplo:

Este div tiene un borde inferior de 5px punteado azul verdoso.
border-bottom: 5px dotted #4ecdc4;

margin:

Definición: Establece el margen exterior de un elemento. Es el espacio entre el borde del elemento y los elementos adyacentes.

Sintaxis: margin: [top] [right] [bottom] [left]; o margin: [valor]; para todos los lados

Ejemplo:

Este div tiene márgenes diferentes: 20px arriba, 10px derecha, 30px abajo, 40px izquierda.
margin: 20px 10px 30px 40px;

padding:

Definición: Establece el relleno interior de un elemento. Es el espacio entre el contenido y el borde del elemento.

Sintaxis: padding: [top] [right] [bottom] [left]; o padding: [valor]; para todos los lados

Ejemplo:

Este div tiene padding diferente: 25px arriba, 15px derecha, 10px abajo, 35px izquierda.
padding: 25px 15px 10px 35px;

vertical-align:

Definición: Establece la alineación vertical de un elemento en línea o celda de tabla respecto a la línea base.

Valores comunes: baseline, top, middle, bottom, text-top, text-bottom

Ejemplo:

Texto normal texto superior texto inferior
vertical-align: top; vertical-align: middle; vertical-align: bottom;

text-indent:

Definición: Establece la sangría de la primera línea de un bloque de texto.

Sintaxis: text-indent: [valor en px, em, %, etc.];

Ejemplo:

Este párrafo tiene una sangría de 50px en la primera línea. Como puedes ver, solo la primera línea está indentada, mientras que las líneas siguientes mantienen su alineación normal.
text-indent: 50px;

text-wrap:

Definición: Controla cómo se ajusta el texto dentro de un elemento. Determina si el texto debe ajustarse a nuevas líneas.

Valores: wrap (por defecto), nowrap, balance

Ejemplo con wrap (normal):

Este texto se ajusta normalmente cuando alcanza el borde del contenedor y continúa en la siguiente línea.
text-wrap: wrap;

Ejemplo con nowrap:

Este texto no se ajusta y se corta cuando alcanza el borde del contenedor.
white-space: nowrap;

text-stroke:

Definición: Añade un contorno al texto (principalmente soportado con prefijo -webkit-). Permite crear efectos de texto con borde.

Sintaxis: -webkit-text-stroke: [ancho] [color];

Ejemplo 1 - Texto transparente con borde:

TEXTO CON BORDE
-webkit-text-stroke: 2px #ff6b6b;
-webkit-text-fill-color: transparent;

Ejemplo 2 - Texto con relleno y borde:

Texto con borde y relleno
-webkit-text-stroke: 1px #333;
-webkit-text-fill-color: #ffe66d;

Notas importantes:

  • border-top y border-bottom son propiedades abreviadas muy útiles para estilizar bordes específicos.
  • margin y padding son fundamentales para el espaciado y layout de elementos.
  • vertical-align solo funciona en elementos en línea y celdas de tabla.
  • text-indent es útil para dar formato a párrafos, especialmente en textos largos.
  • text-wrap es una propiedad relativamente nueva, usar white-space para mayor compatibilidad.
  • text-stroke requiere prefijo -webkit- y tiene limitado soporte en navegadores.
